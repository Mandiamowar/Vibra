# Vibra Pay

**El ecosistema de pagos del futuro: rápido, seguro, ecológico y justo.**

Vibra Pay es una red descentralizada de pagos que utiliza **grafos temporales** y **consenso PoIU**.

## Características

- Velocidad: <1s
- Comisión: 0.1%
- Seguridad biométrica
- Bajo consumo energético

## Instalación

```bash
cd backend
pip install -r requirements.txt
python run.py
Documentación

# Vibra Pay - Whitepaper

## Introducción
Vibra Pay es un ecosistema de pagos descentralizado basado en grafos temporales y consenso PoIU.

## Problema
- Altas comisiones de Visa/Mastercard
- Lentitud de Bitcoin/Ethereum
- Centralización

## Solución
- Grafo temporal sin bloques
- Consenso con validadores e IA
- Firma conductual
- Comisión del 0.1% con quema

## Tokenomics
- 40% de comisiones quemadas
